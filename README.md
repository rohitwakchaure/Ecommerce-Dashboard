# 📌 Project Overview

The **Ecommerce Analytics Dashboard** is an interactive business intelligence solution developed using **Power BI and Microsoft Fabric**.

This project analyzes ecommerce platform data including **orders, customers, payments, tickets, and devices** to generate meaningful insights for business decision-making.

The dashboard enables stakeholders to monitor **sales performance, customer behavior, order trends, and support ticket resolution status** in a single view.

---

# 🎯 Business Objectives

- 📊 Analyze total orders and revenue performance  
- 👥 Track customer distribution and gender analysis  
- 💳 Evaluate payment method usage  
- 🌍 Analyze location-wise order revenue  
- 📱 Identify device usage trends  
- 🎫 Monitor customer support ticket resolution status  
- 📅 Understand order trends over time
